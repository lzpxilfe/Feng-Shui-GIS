# Feng Shui GIS v0.1.0

## 한국어

### 개요
- 이 플러그인은 DEM 기반으로 풍수 입지 후보를 **연구용으로 정량화**합니다.
- 자동 결과는 최종 판정이 아니라, 고고학/역사지리 연구를 위한 1차 후보 지도입니다.

### 주요 기능
- 후보지 점수화: `후보지 포인트 + DEM (+선택 수계)`
- DEM 용어 추출: `혈/명당/조종산/주산/내외청룡/내외백호/수구/입수/안산/조산/미사`
- 지역/시대 컨텍스트:
  - 지역: `동아시아 기본`, `한국`, `중국`, `일본`, `류큐/오키나와`
  - 시대: `고대`, `중세`, `근세`, `근현대`
- 시각화:
  - 용어 포인트 레이어: 굵은 점 심볼
  - 용어 연결선 레이어: 굵은 선 심볼
- 아이콘: `yingyang.png` 사용

### 출력 레이어
- 점수 레이어 `*_fengshui`
  - `fs_culture`, `fs_period`, `fs_model`, `fs_score`, `fs_note` 등
- 용어 포인트 레이어 `*_fengshui_terms`
  - `term_id`, `term_ko`, `score`, `culture`, `period`
- 용어 연결선 레이어 `*_fengshui_terms_links`
  - `term_id`, `term_ko`, `score`, `culture`, `period`

### 연구 엄밀성 원칙
- 문헌 근거와 지형 지표를 연결한 규칙 기반 모델입니다.
- 지역/시대별 파라미터는 연구 가설 비교를 위한 설정이며, 현장 데이터로 검증/보정해야 합니다.
- 상세 근거:
  - `docs/research_matrix.md`
  - `docs/context_profiles.md`
  - `docs/regional_period_notes.md`

### 현대 활용 메모
- 풍수는 오늘날에도 일부 부동산/건축/입지 의사결정에서 영향이 보고됩니다.
- 본 도구는 그런 의사결정을 무비판적으로 따르기 위한 것이 아니라, **근거를 드러내고 비교 가능한 형태로 분석**하기 위한 연구 도구입니다.

## 中文

### 概要
- 本插件将风水选址要素转为 DEM 可计算指标，服务于考古与历史地理研究。
- 自动结果是候选层，不是最终定论。

### 主要功能
- 点位评分：`候选点 + DEM (+可选水系)`
- DEM 术语提取：`穴/明堂/祖宗山/主山/内外青龙/内外白虎/水口/入水/案山/朝山/眉砂`
- 区域与时代上下文：
  - 区域：`东亚基准`, `韩国`, `中国`, `日本`, `琉球/冲绳`
  - 时代：`古代`, `中世`, `近世`, `近现代`
- 可视化：
  - 术语点层：粗点样式
  - 术语连线层：粗线样式
- 图标：使用 `yingyang.png`

### 输出
- 评分层 `*_fengshui`
- 术语点层 `*_fengshui_terms`
- 术语连线层 `*_fengshui_terms_links`

### 研究规范
- 采用“文献依据 + 地形指标”的规则模型。
- 区域/时代参数用于可检验假设，需结合实地与样本数据校准。

## 日本語

### 概要
- 本プラグインは、風水立地の概念を DEM 指標に変換し、考古学・歴史地理研究に利用できる形で可視化します。
- 自動出力は候補レイヤーであり、最終判断そのものではありません。

### 主な機能
- 候補地点スコアリング：`候補ポイント + DEM (+任意の水系)`
- DEM 用語抽出：`穴/明堂/祖宗山/主山/内外青龍/内外白虎/水口/入水/案山/朝山/眉砂`
- 地域・時代コンテキスト：
  - 地域：`東アジア基準`, `韓国`, `中国`, `日本`, `琉球/沖縄`
  - 時代：`古代`, `中世`, `近世`, `近現代`
- 可視化：
  - 用語ポイント層：太いポイント記号
  - 用語リンク層：太いライン記号
- アイコン：`yingyang.png`

### 出力
- スコア層 `*_fengshui`
- 用語ポイント層 `*_fengshui_terms`
- 用語リンク層 `*_fengshui_terms_links`

### 研究上の注意
- 文献根拠に基づくルールベースモデルです。
- 地域/時代パラメータは仮説比較のための設定であり、検証データで補正が必要です。
